# 🚇 交通指南：羽田機場 (HND) ↔️ 龜有

**核心策略**：搭乘 **東京單軌電車 (Tokyo Monorail)** 直達濱松町，轉乘 **JR山手線** 至西日暮里，接著轉乘 **東京地鐵千代田線**（直通JR常磐線），順暢抵達龜有。這條路線雖然稍須進入市區，但班次密集且轉乘標示明確。

---

## 🟢 去程 (機場 ➜ 龜有)
**適用時機**：Day 1 抵達日本，前往龜有大會合時。
- **總車程**：約 1 小時 5 分鐘
- **轉乘次數**：2 次
- **預估車資**：約 900～1,000 日圓

### 詳細路線：
1. **羽田機場起點**：從第三航廈循指標至「東京單軌電車」乘車處。
2. 🚊 **第一段：東京單軌電車 (普通)**
   - 從羽田機場第三航廈搭乘，前往 **單軌電車濱松町站**。
   - *(注意：約 19 分鐘，抵達終點站後需出站轉乘 JR)*
3. 🔄 **轉乘站 1：濱松町站**
   - 步行轉乘 **JR山手線** (往東京 / 上野方面，內環線)。
   - *(注意：搭乘約 20 分鐘，前往西日暮里)*
4. 🔄 **轉乘站 2：西日暮里站**
   - 下車後，轉乘 **東京地鐵千代田線** (往取手 / 綾瀨 / 我孫子方向)。
   - *(注意：千代田線在此直通運轉 JR 常磐線各停，可直接搭到龜有不用下車)*
5. 🏁 **終點：龜有站**
   - 恭喜抵達！先去北口或 Ario 找大型置物櫃寄放行李吧。
